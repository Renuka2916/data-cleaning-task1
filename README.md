# data-cleaning-task1
Project Description:

This project focuses on cleaning and structuring a raw dataset using Microsoft Excel. The main goal was to convert an uncleaned dataset into a clean, readable, consistent, and reliable format for analysis while preserving the original data integrity.

Tool Used:
Microsoft Excel

Cleaning Steps Performed :

-Adjusted row height and column width to ensure proper visibility of all rows and columns.

-Removed unnecessary text present inside brackets using the Replace feature.

-Converted column letters/text into lowercase for consistency in client column.

-Used the TRIM formula to remove extra spaces from text values.

-Used the PROPER formula to make text values follow a consistent case format.

-Separated department name and department location by creating a new table.

-Used the Text to Columns feature to split combined department information into separate columns.

-Checked for duplicate records and removed them using the Remove Duplicates feature.

-Replaced blank cells with NA to clearly represent missing values.

-Used the IFERROR function to handle errors generated due to operations between text and numbers.

-Replaced error values with NA to keep the dataset clean and understandable.

Files Included:

original uncleaned dataset.xlsx :- Original dataset before cleaning

cleaned dataset.xlsx :- Final dataset after cleaning

Final Output:

The final dataset is clean, structured, readable, and ready for further analysis.
